会计科目

内容反映横向联系
级次反映纵向联系
会计科目的级次不同决定了信息使用者的不同

会计等式的基本概念

期初资产 = 期初负债 + 期初所有者权益
会计期间
期末资产 = 期末负债 + 期初所有者权益 + 收入 - 费用
利润合并
期末资产 = 期未负债十期未所有者权益

对资产拥有所有权和索偿权即是权益的表现

静态会计等式

1.全资拥有，资产总额 = 所有者权益总额
2.投资外有负债，资产总额 = 所有者权益总额 + 负债总额

债权人与所有者的区别：
1.性质不同
2.权利不同
3.偿还期限不同
4.风险和收益不同

复式记账

特点：
1.相等的金额
2.两个或两个以上有关账户中
3.进行相互联系的记录

借贷记账

主要特点：
1.记账符号
2.记账规则：借贷相等
3.保持平衡关系
余额计算等式：

  • 资产类：期初金额 + 本期借方发生额 = 本期贷方发生额 + 期末余额
  • 负债、所有者权益、利润类：期初余额 + 本期贷方发生额 = 本期借方发生额 + 期末余额

延伸特点：
不要求固定分类，可能存在资产和负债共同性质的账户。
例如：
应收账款/应付职工薪酬/应交税费
总结：

  • 余额在借方属于资产类账户，余额出现在贷方属于负债类账户。
  • 收入和费用没有余额，因为余额在会计期末统一转到本年利润进行损益核算，以达到查看当期经营成果的目的。

收入和费用的会计等式：
资产 + 费用 = 负债 + 所有者权益 + 收入

会计分录

基本要素：
1.账户(会计科目)其所属明细账户名称
2.内记账方向（记账符号）
3.业务发生额

完整会计循环（记账全过程）

期初余额会计分录（实际业务中在记账凭证登记）每笔分录登入相应的账户结账（每个账户本期的借贷方发生额和期末余额)编制相应的会计报表（余额填入资产负债表；损类账户的本期借贷方发生额填入利润)
